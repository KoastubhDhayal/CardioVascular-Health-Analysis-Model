# CardioVascular-Health-Analysis-Model
This project is a Machine Learning based web application designed to predict the risk of cardiovascular disease based on user health data. The system processes medical and lifestyle data, trains machine learning models, and provides risk predictions through a user-friendly web interface.  
The project integrates data preprocessing, model training, database interaction, and visualization to create a complete end-to-end healthcare analytics pipeline.

Features

Predict cardiovascular disease risk using machine learning

User data entry and search functionality

Data preprocessing and cleaning pipeline

Interactive web interface

Risk prediction visualization

Database connectivity for storing and retrieving user data

Analytical graphs for health trends
